# CryptoFolio Demo Guide

## EN - 3 Minute Flow
1. Positioning (20s): frontend-first portfolio intelligence showcase.
2. Dashboard (70s): value, pnl, drawdown, concentration, VaR.
3. Transactions + Alerts (50s): filters, rule trigger readability.
4. Demo mode + i18n (40s): load demo, switch language, restore pre-demo.

## EN - 5 Minute Flow
| Step | Page | Expected signal | Talk track |
|---|---|---|---|
| 1 | Dashboard | Non-zero KPI and sparkline | Explain state orchestration and derived metrics |
| 2 | Assets | Data-dense table rows | Explain cost basis and 24h contribution |
| 3 | Transactions | Filter results update | Explain interaction model and deterministic data flow |
| 4 | Alerts | Rule list + history | Explain risk explainability and guardrails |
| 5 | Settings | Demo mode + locale + restore | Explain demo reliability and interview reproducibility |

## 中文 - 3 分钟流程
1. 项目定位（20秒）：强调前端工程深度与可靠性。
2. Dashboard（70秒）：讲总资产、盈亏、回撤、集中度、VaR。
3. 交易与告警（50秒）：讲筛选、规则触发和可解释性。
4. Demo 与中英切换（40秒）：加载演示数据、切换语言、恢复现场状态。

## 中文 - 5 分钟流程
| 步骤 | 页面 | 预期结果 | 讲解重点 |
|---|---|---|---|
| 1 | Dashboard | KPI 非零，曲线可读 | 状态编排与指标推导 |
| 2 | Assets | 表格有数据密度 | 成本法、收益与仓位解读 |
| 3 | Transactions | 筛选实时生效 | 交互与数据一致性 |
| 4 | Alerts | 规则与历史可追踪 | 风险解释与告警闭环 |
| 5 | Settings | Demo 模式、语言切换、恢复备份 | 演示可复现与工程可信度 |
