# Demo Script

## 1. 开场（20 秒）

- 这是一个前端实现的加密资产分析与风控看板。
- 核心目标是把“资产查看”升级为“收益分析 + 风险预警”。

## 2. 资产与价格（40 秒）

- 新增 2~3 个资产，输入数量和成本价。
- 展示实时价格、24h 涨跌、总资产和总盈亏。

## 3. 交易与成本法（60 秒）

- 录入买入/卖出交易。
- 切换“加权平均法 / FIFO”，观察总成本和盈亏变化。
- 说明：两种方法导致持仓成本不同，适用于不同复盘口径。

## 4. 收益曲线与风险（60 秒）

- 展示收益曲线（本地快照）。
- 讲解风控卡片：最大回撤、Top1/Top3 集中度、样本波动率。

## 5. 告警中心（40 秒）

- 新建一个价格或仓位告警。
- 刷新数据后，触发条件满足时出现 Toast 并写入告警历史。

## 6. 工程能力（40 秒）

- 本地持久化：资产、交易、快照、告警规则、告警历史。
- 容错：localStorage 脏数据保护。
- 测试：核心公式脚本测试。
- 性能：刷新防重入、请求节流、成本计算缓存。
